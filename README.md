# ⏰ ClockAndWeather

Минималистичный виджет часов и погоды для Windows.

## ✨ Возможности
- 🕐 Часы с секундами (12/24 формат)
- 🌤️ Погода без API-ключей (Open-Meteo)
- 🎨 Две темы: Стандартная + EVE Online
- 📐 Мини-режим (300×50px)
- 🔆 Настраиваемая прозрачность
- 🌍 Автоопределение города
